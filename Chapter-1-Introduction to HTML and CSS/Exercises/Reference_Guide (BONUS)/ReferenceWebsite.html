Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

TYPES OF CSS:

  • Inline Styles: Applied directly to specific HTML elements using the HTML style attribute.
  • Embedded Styles: Defined within the <style> element in the <head> section of the HTML document, applying styles to the entire webpage.
  • External Styles: Defined in separate text files with a .css file extension and linked to the HTML document using the <link> element in the <head> section.
  • Imported Styles: Similar to External Styles, where styles are defined in separate CSS files, but they are imported into another CSS file using the @import rule.

CSS SYNTAX:

Style sheets are composed of "Rules" that describe the styling to be applied. Each Rule contains a Selector and a Declaration

COMMON FORMATTING PROPERTIES:

  • background-color: Specifies the background color of an element.
  • color: Sets the text color of an element.
  • font-family: Defines the font family or typeface to be used for text.
  • font-size: Sets the size of the font for text.
  • font-style: Determines the style of the font, such as normal, italic, or oblique.
  • font-weight: Specifies the thickness or boldness of the font.
  • line-height: Sets the height of a line of text, affecting the spacing between lines.
  • margin: Specifies the margin around an element, controlling the space between the element's border and surrounding elements.
  • text-align: Aligns the text horizontally within its container, such as left, right, center, or justified.
  • text-decoration: Adds decoration to text, such as underline, overline, line-through, or none.
  • width: Sets the width of an element, controlling its horizontal size.
