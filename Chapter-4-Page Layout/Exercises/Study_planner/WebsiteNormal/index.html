STUDY PLANNER


This comprehensive study planner is meticulously crafted to guide you through the intricacies of the Web Development course with ease and efficiency. It's tailored to ensure that you gain a deep understanding of web development principles and techniques, allowing you to build websites with confidence and proficiency. Through structured repetition and hands-on practice, you'll sharpen your skills, reinforce your knowledge, and accelerate your learning curve in this dynamic field.


HTML :


Timings Explanation
4:00 - 5:00 Practice and learn HTML code from sololearn.
5:00 - 6:00 Do light warm up by coding a simple website. Even though this is just a warm up, you should employ proper techniques for consistency.
6:00 - 5:00 Make the first 3 website without any styling. Focus solely on coding in HTML.
6:00 - 6:30 Take a light break.
6:30 - 7:00 Make the last 3 websites than make sure all your websites are using semantically correct elements before proceeding with tomorrows exercise.



CSS :


Timings Explanation
4:00 - 5:00 Practice and learn CSS code from sololearn.

6:30 - 7:00 Take a break for 30 minutes.
